# {{main/ns}}

## quickstart

> this guide will start a bb repl.  it's expected that all the interactions
> with this project are done through the repl.

- move into the project root
  ```command
  cd ~/path/to/{{main/ns}}
  ```
- run a file
  ```command
  bb src/{{main/ns}}
  ```
- run a bb repl
  ```command
  bb socket-repl 1667
  ```
